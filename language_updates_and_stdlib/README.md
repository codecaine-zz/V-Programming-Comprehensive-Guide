# Language updates and standard library

This folder collects the newer language features and the standard library examples that complement the main tutorial. To make the material easier to explore, it is now grouped by purpose rather than by one long flat list.

## Sections

- [01_language_basics_updates](01_language_basics_updates) — newer language features such as options/results, generics, interfaces, sum types, attributes, and directives.
- [02_standard_library](02_standard_library) — practical standard library examples organized into focused sections such as text and data, system and files, networking and web, and CLI/platform topics.

## Suggested reading order

1. Start with the language basics updates to understand the modern V features.
2. Move into the standard library sections that match the kind of program you want to build.
3. Use the examples as reference material when you need a small, focused pattern for a real project.
